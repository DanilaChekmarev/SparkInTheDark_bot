ForeignCurrencyMarket.cmd_ForeignCurrencyMarket ( Message message)

Обработчик команды /ForeignCurrencyMarket.

Эта асинхронная функция обрабатывает команду /ForeignCurrencyMarket, запрашивает данные о валютном рынке из API центробанка и отправляет пользователю котировки валют за последнюю зарегистрированную дату.

ForeignCurrencyMarket.combine_in_a_single_message ( ForeignCurrencyMarketData)

Объединяет данные о валютном рынке в одно сообщение.

Эта функция принимает данные о валютном рынке в формате словаря и формирует строку, содержащую дату и информацию о каждой валюте, включая название, стоимость, номинал и код.
